A full microservices application built with Spring Boot consisting of a User Service (registration, JWT authentication, email verification, password reset, and profile management), a Todo Service (full CRUD with soft delete, priority and category tracking, and JWT-based ownership), a Notification Service (HTML email and SMS delivery with logging for verification, welcome, password reset, todo reminders, and overdue alerts), a Config Server for centralized configuration, and a Eureka Server for service discovery. Features scheduled cron jobs that automatically send SMS and email notifications to users based on todo priority and due date status.
